Данный проект является упрощенной информационной системой для автоматизации процессов составления расписания в учебном заведении,
а также его просмотра.

1.Требования к ПО:
- СУБД transact-sql (Sql server)
- Visual Studio или другая IDE c платформой .NET 6.0 и выше

2.Алгоритм установки:
- Скачайте репозиторий.
- Перейдите в терминале в проект.
- Запустите команду -dotnet restore для установки зависимостей
- Создайте базу данных transact-sql, измените файл подключения к бд (appsettings.json)
- Выполните команду -dotnet ef database update для применения миграций бд
- Выполните сборку проекта и запустите.
